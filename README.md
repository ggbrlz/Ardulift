# ARDULIFT

ARDULIFT é um projeto de elevador didático com Arduino, desenvolvido para demonstrar conceitos de robótica, automação e integração entre hardware e software.

## Objetivo
Simular o funcionamento de um elevador com acesso a 4 andares, de forma interativa e educativa.

## Tecnologias e componentes
- Arduino
- Servo motor de rotação contínua
- Push buttons
- Display LCD
- Módulo I2C
- Protoboard

## Como funciona
O usuário seleciona um dos andares por meio de botões. O sistema movimenta a cabine e informa no display o andar atual e o deslocamento.

## Desafios do projeto
Um dos motores inicialmente testados não apresentou força suficiente para levantar a cabine, então a solução foi adaptada para um servo motor de rotação contínua. Também foi criado um modo de configuração manual para ajuste do posicionamento da cabine.

## Contexto
Projeto desenvolvido no curso técnico em Informática e apresentado no evento IF de Portas Abertas.
